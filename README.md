## 💻 How to Run the Code

Since all solutions are written in standard **C++**, you can easily compile and run any file using `g++` (the GNU C++ compiler).

### 1. Navigate to the specific topic folder
```bash
cd 08_Dynamic_Programming
```

### 2. Compile the C++ file
*It is recommended to compile using C++17 or higher.*
```bash
g++ -std=c++17 filename.cpp -o output
```

### 3. Run the executable
**Mac/Linux:**
```bash
./output
```
**Windows:**
```bash
output.exe
```
